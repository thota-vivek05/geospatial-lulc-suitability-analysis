# Geospatial Suitability Analysis Web Application

This folder contains the frontend web application and machine learning scripts used for visualizing and analyzing geospatial suitability outputs.

---

# Features

- Interactive suitability map visualization
- Raster and vector layer rendering
- Spatial analysis outputs
- Machine learning integration
- GIS-based visualization workflow

---

# Technologies Used

## Frontend

- Vite
- JavaScript
- HTML
- CSS

## Geospatial Tools

- QGIS
- GDAL
- Raster Analysis

## Machine Learning

- Python
- Scikit-learn
- Rasterio
- NumPy

---

# Machine Learning Scripts

## `train_random_forest_suitability.py`

Implements Random Forest based suitability prediction workflows.

## `train_multilayer_spatial_random_forest.py`

Implements advanced multilayer spatial suitability modeling using machine learning techniques.

---

# Project Workflow

1. Raster preprocessing
2. Spectral index generation
3. Spatial factor extraction
4. Suitability analysis
5. Machine learning prediction
6. Web visualization
